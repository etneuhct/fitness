# Project: Fitness
Ce projet crée des outils enregistrant l'activité physique.


# Applications

- [Fitness API](https://github.com/etneuhct/fitness_api)

Serveur Django récupérant des données d'activités physiques. Essentiellement lié à un controller pour monitorer l'activité d'un vélo stationnaire
- [Fitness Client](https://github.com/etneuhct/fitness_client)

Client Angular affichant les stats et la progression de l'exercice en cours
- [Fitness Bike Controller](https://github.com/etneuhct/fitness_bike_controller)

Micro-controller connecté au vélo qui intercepte l'activité et transmet au serveur.
